# Calories Burn Prediction using Random Forest

Machine learning regression project for predicting calories burned using Random Forest Regression in R.

## Overview
This project develops a Random Forest Regression model to predict calories burned based on physiological and exercise-related features such as age, height, weight, duration, heart rate, body temperature, and gender.

## Features
- Data cleaning and preprocessing
- Train-test data splitting
- Random Forest Regression modeling
- Model evaluation using regression metrics
- Actual vs predicted analysis
- Residual analysis
- Feature importance analysis

## Technologies Used
- R
- randomForest
- caTools
- Metrics
- ggplot2
- dplyr

## Dataset
The dataset contains 15,000 records with physiological and exercise-related features used to predict calories burned.

## Evaluation Metrics
- RMSE: 3.04
- MAE: 1.88
- R²: 0.9976

## Results
The Random Forest Regression model achieved strong predictive performance and identified important factors influencing calories burned during physical activity.
